# bats
utility tool for creating xcopy batch files

## usage:
1. add paths to remote files in *filelist.txt*
2. set the remote and local variables in *config.ini* (note: paths here and in the filelist are case sensitive)
3. run *bats.py*
4. run *soup.bat*

**run default profile**
```bash
bats.py
```
**run with profile**
```
bats.py -p [profile]
```
